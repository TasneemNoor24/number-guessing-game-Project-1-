# 🎮 Number Guessing Game (Java Swing)

This is a simple GUI-based Number Guessing Game built using Java Swing.  
The player has to guess a randomly generated number within a limited number of attempts based on difficulty level.

---

## 📌 Features
- 🎯 Three difficulty levels (Easy, Medium, Hard)
- 🔢 Random number generation
- ⏳ Limited attempts system
- 📊 Score and High Score tracking
- 🖥️ Interactive GUI using Java Swing (JFrame)
- 📉 Hints like "Too High" or "Too Low"

## 🛠️ Technologies Used
- Java
- Swing (JFrame, JButton, JLabel, JTextField)
- Object-Oriented Programming (OOP)
